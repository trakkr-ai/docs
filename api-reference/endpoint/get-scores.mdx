## Endpoint

```
GET https://api.trakkr.ai/get-scores
```

## Headers

| Header          | Type   | Required | Description                                          |
| --------------- | ------ | -------- | ---------------------------------------------------- |
| `Authorization` | string | Yes      | Your API key in Bearer format: `Bearer YOUR_API_KEY` |
| `Content-Type`  | string | Yes      | Must be `application/json`                           |

## Query Parameters

| Parameter    | Type   | Required | Description                                                                                                |
| ------------ | ------ | -------- | ---------------------------------------------------------------------------------------------------------- |
| `brand`      | string | Yes      | The brand ID (32-character string in format: `0000000000000x000000000000000000`)                           |
| `breakdown`  | array  | No       | Array of breakdown types. Valid values: `["model"]`, `["prompt"]`, or `["model", "prompt"]`. Default: `[]` |
| `start_date` | string | No       | Start date in YYYY-MM-DD format. If not provided, defaults to 28 days before end_date                      |
| `end_date`   | string | No       | End date in YYYY-MM-DD format. If not provided, defaults to the most recent report date                    |

## Response Format

The response is a JSON object containing:

| Field        | Type   | Description                                    |
| ------------ | ------ | ---------------------------------------------- |
| `brand_id`   | string | The brand ID that was queried                  |
| `start_date` | string | The start date used for the query (YYYY-MM-DD) |
| `end_date`   | string | The end date used for the query (YYYY-MM-DD)   |
| `breakdown`  | array  | The breakdown types that were requested        |
| `data`       | array  | Array of daily report data                     |
| `errors`     | array  | Array of error messages (if any)               |

### Data Array Structure

Each item in the `data` array represents a daily report with:

| Field                    | Type   | Description                                        |
| ------------------------ | ------ | -------------------------------------------------- |
| `date`                   | string | The report date (YYYY-MM-DD)                       |
| `visibility`             | number | Brand visibility score (0.00 to 100.00)            |
| `presence`               | number | Brand presence score (0.00 to 100.00)              |
| `models`                 | array  | Model breakdown data (if `"model"` in breakdown)   |
| `prompts`                | array  | Prompt breakdown data (if `"prompt"` in breakdown) |
| `model_prompt_breakdown` | array  | Combined model/prompt data (if both in breakdown)  |

## Notes

- **Date Range**: Maximum 90 days between start_date and end_date
- **Default Date Range**: If no dates provided, defaults to the last 28 days ending on the most recent report date
- **Access**: Your API key must have access to the specified brand
- **Rank Values**: In model/prompt breakdown, rank can be `null` if the brand doesn't appear in that specific combination
